# Shared Format Elements — 共享格式元素

本文件定义两种输出格式（结构化/廷议）共用的视觉元素和模板片段。各格式模板引用本文件而非各自重复定义。

---

## 视觉设计规范

### 分隔线体系

| 用途 | 符号 | 说明 |
|------|------|------|
| 轮次之间主分隔 | `---` | markdown hr |
| 轮次标题卡 | `┌─────────────────────────────────────┐` 单线框 | 宽度统一 37 字符 |
| 裁决/绝招等特殊标题卡 | 同上单线框 | 统一 |
| 开场仪式大标题 | `╔═════════════════════════════════════╗` 双线框 | 宽度统一 37 字符 |
| 步骤结束提示 | `---` + 引用块 | 保持不变 |

### 进度条格式

所有进度条统一为 16 格：

```
[████░░░░░░░░] 第1轮/共3轮 — 群雄立论
[████████░░░░] 第1轮完成/共3轮
[████████████░░] 第2轮/共3轮 — 短兵相接
[████████████████] 第2轮完成/共3轮
[████████████████] 第3轮/共3轮 — 峰回路转
```

### Emoji 规范

- 保留功能型：🔴🔵⚪阵营色、📊进度、⏸️暂停、🔥绝招/争议
- 去掉装饰型堆叠
- 金句标签：`✨ 金句`（不含"本轮"）

### 信心仪表盘

采用纯数字 + 趋势箭头，避免条形图对齐问题：

```
📊 信心趋势：
成员      R1   R2   R3   趋势
{成员A}    7    5    3    ↓↓↓
{成员B}    8    8    7    →→
{成员C}    ?    ?    5    ↑↑
```

**规则：**
- 用纯数字展示信心值，标注变化趋势（↑↑/↓↓↓/→→）
- 被用户投票的成员标注 🗳️
- 观察者前两轮用 `?` 表示隐藏信心，第3轮揭示
- 信心值变化必须与辩论内容一致

```
{绝招效果详细描写，结合当前辩论内容}

```

## 通用交互格式

### 金句提取

每轮自动提取1-2条最佳发言片段：

```
✨ 金句
> "{金句原文}"
> — {成员名} [{场景标签，如"反击马斯克"/"总结陈词"}]
```

### 观众反应

在成员发言结束后，随机插入虚拟观众反应（每轮1-2次，不喧宾夺主）：

```
（观众席传来低声议论……）
（观众席一阵骚动——有人鼓掌，有人摇头）
（角落里传来一声"好！"）
（观众席安静了一瞬……）
（有观众掏出了手机开始录屏……）
（观众席爆发出笑声）
（观众席倒吸一口冷气……）
```

**规则：**
- 反应类型要与发言内容匹配（精彩论点→鼓掌，惊人言论→骚动，幽默→笑声）
- 用括号呈现，轻量不干扰阅读
- 每轮最多2-3次，保持克制
- 不要每次发言后都加反应，要有留白
