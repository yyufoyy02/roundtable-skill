# 05 - 决策记录与行动

## 一、重大决策时间线

### 1.1 鄞县实验（1047-1050年）
- **决策**：在鄞县推行水利建设、贷谷于民、兴办学校
- **逻辑**：从基层实践中验证改革思路
- **结果**：政绩显著，青苗法的雏形在此形成
- **意义**：这是王安石改革思想的第一次实践验证

### 1.2 拒绝多次升迁（1050-1060年代）
- **决策**：多次推辞朝廷升迁机会，坚持在地方任职
- **逻辑**：不急于求官，先积累实际经验
- **结果**：文名远播，最终被神宗破格提拔
- **意义**：体现了王安石"经世致用"的价值观——重实践轻虚名

### 1.3 上万言书（1058年）
- **决策**：向仁宗皇帝上《言事书》
- **逻辑**：系统阐述改革主张，争取最高权力支持
- **结果**：未获重视
- **事后反思**：十年后成为熙宁变法的理论纲领，说明时机比内容更重要

### 1.4 熙宁变法的启动（1069-1070年）
- **决策**：就任参知政事，全面启动变法
- **逻辑**：时机成熟——神宗年轻有为、锐意改革
- **具体措施**：
  - 设立制置三司条例司（改革指挥中心）
  - 青苗法（1069年）——抑制高利贷
  - 免役法（1071年）——改革差役制度
  - 市易法（1072年）——打破商业垄断
  - 保甲法（1070年）——寓兵于农
  - 均输法——改革物资调配
  - 方田均税法——重新丈量土地
  - 科举改革——废除诗赋取士

### 1.5 对反对派的回应
- **决策**：坚决不妥协，但不用公权力报私仇
- **逻辑**："辟邪说，难壬人，不为拒谏"
- **具体行为**：
  - 驳斥司马光：写《答司马谏议书》
  - 对苏轼：虽反对其政见，但在乌台诗案时上书营救
  - 对欧阳修：虽政见分歧，但写了情真意切的祭文

### 1.6 两次罢相（1074年、1076年）
- **第一次罢相（1074年）**：
  - 触发：郑侠《流民图》+ 旱灾 + 朝野压力
  - 王安石的反应：一度动摇但最终坚持
  - 复相（1075年）：短暂回归
- **第二次罢相（1076年）**：
  - 触发：新党内部分裂 + 神宗态度变化
  - 王安石的反应：彻底退出朝堂
  - 退居江宁钟山，不再过问政治

### 1.7 退隐后的决策
- **不参与政治**：虽退隐但不再复出
- **潜心学问**：参佛修禅、著述
- **闻免役法废**：1086年听闻免役法被废除，失声痛哭，不久病逝

## 二、决策模式分析

### 2.1 言行一致 vs 不一致

| 维度 | 一致性 | 分析 |
|------|--------|------|
| 理财观 | 一致 | 始终坚持"开源优先" |
| 人才观 | 部分一致 | 理论上重视人才，但用人实践中重才轻德 |
| 道义观 | 一致 | 始终"度义而后动" |
| 变法决心 | 一致 | 从未放弃变法理念 |
| 对待反对者 | 复杂 | 政治上不妥协，但私交上保持尊重 |

### 2.2 决策失败案例

1. **青苗法的"抑配"**：
   - 设计：低息贷款给农民
   - 实际执行：地方官为完成指标强行摊派
   - 教训：顶层设计再好，执行层面也必须到位

2. **吕惠卿事件**：
   - 决策：重用吕惠卿
   - 结果：吕惠卿在王安石罢相后翻脸
   - 教训：用人不能只看才华，也要看品德

3. **改革节奏**：
   - 决策：同时推行十几个新法
   - 结果：执行层面的人力物力跟不上
   - 教训：系统性改革需要匹配系统性的执行能力

### 2.3 决策启发式提取

1. **先做实验再推广**：鄞县实验 → 全国推广
2. **时机比内容更重要**：万言书被仁宗忽视，被神宗采纳
3. **需要最高权力支持**：没有神宗的信任，变法不可能推进
4. **执行是设计的10倍重要**：所有变法的问题都出在执行层面
5. **用人决定成败**：好制度+坏执行者 = 坏结果
