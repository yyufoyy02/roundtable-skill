# 02 - 对话与即兴思考

## 一、与宋神宗的对话（"越次入对"）

### 1.1 初次召见（1068年）
- 背景：神宗即位后"越次入对"——打破常规直接召见王安石
- 核心对话：
  - 神宗问治国之要，王安石对以"择术为先"
  - 王安石提出"变风俗、立法度"的总体改革方案
  - 神宗问"唐太宗何如"，王安石答："当以尧舜为法"
- **分析**：王安石面对年轻皇帝时不是讨好，而是直接提出最高标准，体现其"法意不法官"——不效法近世（唐太宗），而是效法上古圣王（尧舜）

### 1.2 变法过程中的奏对
- 关于天变：神宗因彗星出现而犹豫，王安石说"天变不足畏"
- 关于用人：王安石力荐吕惠卿，称"惠卿之贤，岂特今人，虽前世儒者，未易比也"
- 关于反对声音：王安石坚持"辟邪说，难壬人，不为拒谏"

### 1.3 对话模式特征
- **直陈利弊**：不绕弯子，直接说出问题和解决方案
- **引用经典**：用先王之道为行动提供合法性
- **反问引导**：用反问让对方自己得出结论
- **以退为进**：偶尔表达退意（如多次请求辞相），实则迫使对方更加坚定

## 二、与司马光的辩论

### 2.1 答司马谏议书（1070年）
- **背景**：司马光写信批评变法"侵官、生事、征利、拒谏"
- **辩论策略分析**：
  1. 先确立论战基础——"名实之辩"：把争论拉到理性层面
  2. 逐一反驳——四个"不为"的排比结构
  3. 利益分析——"士大夫多以不恤国事、同俗自媚于众为善"
  4. 历史类比——盘庚迁都
  5. 以退为进——"如曰今日当一切不事事...则非某之所敢知"
- **即兴反应特征**：
  - 不攻击对方人品，只论事理
  - 不回避反对声，而是将其纳入论证体系
  - 语气不卑不亢，刚柔并济
  - 结尾是最强硬的表态——看似退让，实则划定不可妥协的底线

### 2.2 理财之辩
- 司马光："天地所生财货百物，止有此数，不在民间则在公家"
- 王安石："善理财者，民不加赋而国用饶"
- **本质分歧**：零和思维 vs 正和思维

### 2.3 改变立场的瞬间
- 面对郑侠《流民图》时，王安石一度动摇（"求退"），但最终坚持变法立场
- 这说明他并非完全不知退让，但在核心原则上绝不妥协

## 三、与其他人物的互动

### 3.1 与欧阳修
- 欧阳修是王安石的举荐人，有师长之谊
- 欧阳修最终成为变法反对者
- 王安石写的《祭欧阳文忠公文》被评"压倒曾巩、苏轼、苏辙"，说明二人虽有政见分歧，但王安石在私交上仍保持尊重

### 3.2 与苏轼
- 苏轼上万言书批评新法弊端
- 王安石对苏轼的才华给予高度评价
- 乌台诗案时，已退居的王安石上书营救苏轼——"岂有盛世而杀才士乎"

### 3.3 与吕惠卿
- 从"情同师徒"到彻底决裂
- 王安石曾高度评价："学先王之道而能用者，独惠卿而已"
- 吕惠卿在王安石罢相后翻脸——暴露王安石用人的重大失误

### 3.4 拒绝回答的问题
- 关于个人享受、物质追求——从不回应，认为这些不是该讨论的
- 关于风水天象——虽然提出"天变不足畏"，但在公开场合不直接否定天命
- 关于私人恩怨——从不利用公权力报私仇

## 四、被追问时的回答方式

| 追问类型 | 王安石的反应 |
|---------|------------|
| 对变法的质疑 | 先重新定义概念（名实之辩），再逐一反驳 |
| 对用人失误的批评 | 承认事实但不改变原则——"人才不足"不等于"不应该改革" |
| 对道德的质疑 | 不辩解私德，只谈公义 |
| 皇帝的犹豫 | 以退为进，反而强化自己的立场 |
| 旧党的和解 | 在原则问题上绝不退让 |
