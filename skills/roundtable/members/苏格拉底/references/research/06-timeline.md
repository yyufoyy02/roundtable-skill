# 06-timeline: 人物时间线

> 调研目标：苏格拉底一生关键节点
> 调研时间：2026-04-14

## 生平时间线

| 时间 | 事件 | 对思想的影响 |
|------|------|------------|
| 约前470年 | 出生于雅典阿罗珀克区 | 父亲 Sophroniscus（雕刻师），母亲 Phaenarete（产婆）。公民身份使他有权参与政治 |
| 约前450-440年代 | 受教育，接触自然哲学家（阿那克萨哥拉）的思想 | 接触前苏格拉底哲学传统，但转向人类事务 |
| 约前440-430年代 | 师从阿基劳斯（Archelaus）？ | 学术界有争议。有记载说他是阿基劳斯的学生 |
| 前432年 | 伯罗奔尼撒战争开始 | 在战争中作为重装步兵服役，以勇敢著称（《会饮篇》中阿尔基比亚德斯描述） |
| 前432-424年 | 参加波提达亚战役、德利翁战役、安菲波利斯战役 | 军事经历展示了他的勇气和纪律——即使在战斗中也保持冷静 |
| 前423年 | 阿里斯托芬《云》公演 | 苏格拉底被讽刺为收费的自然哲学家和无神论者。这为24年后的审判埋下了舆论伏笔 |
| 约前420年代 | 遇见柏拉图（约20岁的柏拉图成为他的学生） | 柏拉图后来成为他最重要的传播者，但也在对话中注入了自己的思想 |
| 约前420年代 | 教导阿尔基比亚德斯、克里同、色诺芬等人 | 学生来自雅典精英阶层——这后来被用来指控他"腐蚀青年" |
| 前416年 | （可能参与）西西里远征相关的政治讨论 | 阿尔基比亚德斯是西西里远征的推动者，苏格拉底曾试图劝阻他 |
| 前415年 | 阿尔基比亚德斯叛逃斯巴达 | 苏格拉底与这位争议学生的关系成为日后被攻击的把柄 |
| 前406年 | 阿吉纽西战役后的"将军审判" | 苏格拉底作为议事会唯一成员，拒绝参与对六位将军的集体非法审判。展示了他的道德勇气 |
| 前404年 | 三十僭主统治时期 | 拒绝执行逮捕萨拉米斯莱昂的命令。再次展示对不公正的拒绝 |
| 前403年 | 民主制恢复，大赦法令 | 雅典恢复民主，对僭主时期的罪行实行大赦。但苏格拉底的批评者仍在积累不满 |
| 前399年 | 被美勒图斯、阿努图斯、吕孔控告 | 控告：不敬神、腐蚀青年。由501人陪审团审判 |
| 前399年 | 发表《申辩篇》辩护 | 被判处死刑（280:220的投票结果） |
| 前399年 | 在狱中拒绝逃跑（《克里同篇》） | 克里同安排了越狱，苏格拉底以"社会契约"为由拒绝 |
| 前399年 | 饮鸩而死（《斐多篇》） | 70岁左右。最后的话："克里同，我们欠阿斯克勒庇俄斯一只公鸡，别忘了还。" |
| 前399年后 | 柏拉图、色诺芬、安提斯泰尼等记录苏格拉底思想 | 苏格拉底的影响通过他的学生传播 |

## 思想影响时间线

| 时间 | 事件 | 影响 |
|------|------|------|
| 前399年后 | 柏拉图建立学园（Academy） | 西方第一个系统性的哲学教育机构 |
| 前4世纪 | 色诺芬撰写《回忆苏格拉底》 | 提供了一个更"日常"的苏格拉底形象 |
| 前4世纪 | 安提斯泰尼创立犬儒学派 | 受苏格拉底的简朴生活影响 |
| 前3世纪 | 斯多葛学派兴起 | 芝诺、克莱安塞斯将苏格拉底的伦理学系统化 |
| 中世纪 | 苏格拉底通过柏拉图影响基督教哲学 | 奥古斯丁的新柏拉图主义 |
| 17-18世纪 | 启蒙运动 | 苏格拉底被视为理性主义和自由思想的先驱 |
| 19世纪 | 黑格尔、克尔凯郭尔等重新诠释苏格拉底 | 克尔凯郭尔特别关注苏格拉底的"反讽" |
| 1872年 | 尼采《悲剧的诞生》 | 对苏格拉底的激进批评——理性主义扼杀生命本能 |
| 20世纪 | 苏格拉底教学法在法学院普及 | 芝加哥大学法学院等 |
| 20世纪 | CBT认知行为疗法采用"苏格拉底式提问" | Aaron Beck 等人 |
| 21世纪 | 苏格拉底在AI伦理、批判性思维教育中的持续影响 | "AI时代我们更需要苏格拉底式追问" |

## 最新动态（2024-2026）

由于苏格拉底是历史人物，没有"最新动态"。但近年来关于他的讨论包括：
- AI时代"苏格拉底式追问"的复兴——面对大语言模型的"幻觉"问题，苏格拉底的质疑精神更加重要
- 数字民主 vs 苏格拉底式精英主义的辩论重新兴起
- "苏格拉底方法"在教育科技中的应用
