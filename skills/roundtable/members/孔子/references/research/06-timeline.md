# 06-人物时间线

> 孔子（前551年—前479年）完整生平时间线

## 关键里程碑

| 时间 | 年龄 | 事件 | 对思想的影响 |
|------|------|------|------------|
| 前551年 | 出生 | 生于鲁国陬邑（今山东曲阜），父叔梁纥（武士），母颜氏（平民） | 出身：贵族后裔但家道中落，"吾少也贱，故多能鄙事" |
| 前549年 | ~3岁 | 丧父，随母颜氏生活，家境贫寒 | 早年困苦塑造了"不耻下问"的学习态度 |
| 前535年 | ~15岁 | "吾十有五而志于学" | 立志向学，确立了终身学习的方向 |
| 前533年 | ~20岁 | 娶亓官氏，生子孔鲤（字伯鱼） | 开始承担家庭责任 |
| 前532年 | ~19岁 | 担任"委吏"（管理仓库的小吏） | 从最基层做起，了解民间疾苦 |
| 前525年 | ~27岁 | 赴洛邑考察周礼，问礼于老子（据《史记》） | 接触道家思想，但选择了不同的道路 |
| 前522年 | ~30岁 | "三十而立"，创办私学 | 开创平民教育，提出"有教无类" |
| 前518年 | ~34岁 | 赴周都洛邑学习礼乐 | 深化对周礼的理解 |
| 前517年 | ~35岁 | 鲁国内乱（三桓之乱），避难齐国 | 亲身体验政治动荡，齐景公问政 |
| 前510年 | ~42岁 | 返回鲁国，继续教学 | "四十而不惑"——思想趋于成熟 |
| 前502年 | ~50岁 | "五十而知天命" | 接受了自己的使命和局限 |
| 前501年 | ~51岁 | 出任鲁国中都宰 | 政治理想的实践开始 |
| 前500年 | ~52岁 | 升任司空、大司寇，夹谷之会外交胜利 | 政治生涯的高峰 |
| 前498年 | ~54岁 | 推动"堕三都"改革 | 政治理想遭遇现实阻力 |
| 前497年 | ~55岁 | 弃官离鲁，开始周游列国 | 人生最大转折——放弃安全感追求理想 |
| 前496年 | ~56岁 | 在卫国被监视，被迫离开 | 各国君主对他的态度：尊重但不采纳 |
| 前492年 | ~60岁 | "六十而耳顺" | 历经磨难后心态趋于平和 |
| 前489年 | ~63岁 | 陈蔡绝粮，弟子动摇 | 精神最低谷，但"弦歌不辍" |
| 前488年 | ~64岁 | 到达楚国，楚昭王欲封地但被阻止 | 最后一次政治机会的丧失 |
| 前484年 | ~68岁 | 返回鲁国 | "七十而从心所欲，不逾矩"——归鲁后思想达到化境 |
| 前483年 | ~69岁 | 独子孔鲤去世 | 晚年丧子之痛 |
| 前482年 | ~70岁 | 最得意弟子颜回去世 | "天丧予！"——精神上最大的打击 |
| 前481年 | ~71岁 | 最忠勇弟子子路在卫国政变中遇难 | 多位核心弟子相继离世 |
| 前479年 | 73岁 | 4月11日去世，葬于曲阜孔林 | 但他开创的儒学和私学传统延续至今 |

## 思想发展轨迹

```
15岁 志于学        → 确立学习方向
30岁 而立          → 教育理念成型，创办私学
40岁 不惑          → 思想体系趋于成熟
50岁 知天命        → 接受使命与局限
55岁 弃官出走      → 理想主义的终极考验
60岁 耳顺          → 经历磨难后心态平和
70岁 从心所欲不逾矩 → 内在自由与外在秩序的统一
```
