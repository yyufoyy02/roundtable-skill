# 01-writings: 诸葛亮著作与系统思考（深度蒸馏版）

> 调研时间：2026-04-14 | 调研者：女娲Agent-1

## 一、著作清单与核心论点

### 1.1 《隆中对》（207年）
**出处**：《三国志·蜀书·诸葛亮传》陈寿撰（一手）
**性质**：战略规划文献，诸葛亮27岁时向刘备提出的天下三分战略

**原文核心段落**：
> "自董卓已来，豪杰并起。曹操比于袁绍，则名微而众寡。然操遂能克绍，以弱为强者，非惟天时，抑亦人谋也。今操已拥百万之众，挟天子而令诸侯，此诚不可与争锋。孙权据有江东，已历三世，国险而民附，贤能为之用，此可以为援而不可图也。"
> "将军既帝室之胄，信义著于四海，总揽英雄，思贤如渴，若跨有荆、益，保其岩阻，西和诸戎，南抚夷越，外结好孙权，内修政理；天下有变，则命一上将将荆州之军以向宛、洛，将军身率益州之众出于秦川，百姓孰敢不箪食壶浆以迎将军者乎？"

**反复出现的核心论点**（≥3次复现→真信念）：
1. **不可与争锋/可以为援而不可图** → 实力评估是行动的前提（天下三分模型）
2. **非惟天时，抑亦人谋** → 人为努力可改变客观条件（以人胜天）
3. **跨有荆益→内修政理→待天下有变→两路北伐** → 分阶段规划（待时而动）
4. **西和诸戎，南抚夷越，外结好孙权，内修政理** → 多维平衡（联盟博弈）

### 1.2 《前出师表》（227年）
**出处**：《三国志·蜀书·诸葛亮传》（一手，公认诸葛亮亲笔）
**性质**：北伐前给后主刘禅的政治遗嘱/奏章

**原文核心段落**：
> "宫中府中，俱为一体，陟罚臧否，不宜异同。若有作奸犯科及为忠善者，宜付有司论其刑赏，以昭陛下平明之理，不宜偏私，使内外异法也。"
> "亲贤臣，远小人，此先汉所以兴隆也；亲小人，远贤臣，此后汉所以倾颓也。"
> "臣本布衣，躬耕于南阳，苟全性命于乱世，不求闻达于诸侯。先帝不以臣卑鄙，猥自枉屈，三顾臣于草庐之中，咨臣以当世之事，由是感激，遂许先帝以驱驰。"

**核心论点**：
1. **陟罚臧否，不宜异同** → 制度公平（法治思想）
2. **亲贤臣，远小人** → 人才是兴衰关键
3. **广开言路，察纳雅言** → 决策要听取多元意见
4. **报先帝而忠陛下** → 忠诚与承诺履行

### 1.3 《后出师表》（228年）
**出处**：三国吴张俨《默记》（真伪有争议，多数现代学者认为系后人整理诸葛亮思想）
**性质**：第二次北伐前的政治宣言

**原文核心段落**：
> "先帝虑汉贼不两立，王业不偏安，故托臣以讨贼也。"
> "然不伐贼，王业亦亡。惟坐而待亡，孰与伐之？"
> "臣鞠躬尽力，死而后已。至于成败利钝，非臣之明所能逆睹也。"

**核心论点**：
1. **汉贼不两立，王业不偏安** → 不妥协的原则立场
2. **坐而待亡孰与伐之** → 以攻代守的战略逻辑
3. **鞠躬尽力，死而后已** → 极致的承诺精神
4. 连举六条"未解" → 严密的论证逻辑（反问式说服）

### 1.4 《诫子书》（234年）
**出处**：《太平御览》引《诸葛亮集》（一手，公认诸葛亮亲笔）
**性质**：临终前写给儿子诸葛瞻（8岁）的家训

**全文**：
> "夫君子之行，静以修身，俭以养德。非淡泊无以明志，非宁静无以致远。夫学须静也，才须学也，非学无以广才，非志无以成学。慆慢则不能励精，险躁则不能冶性。年与时驰，意与日去，遂成枯落，多不接世，悲守穷庐，将复何及！"

**核心论点**：
1. **静以修身，俭以养德** → 内修是外在成就的基础
2. **非淡泊无以明志** → 清心寡欲才能看清目标
3. **非学无以广才，非志无以成学** → 学习与志向的辩证关系
4. **年与时驰** → 时间紧迫感

### 1.5 《诫外甥书》
**出处**：《太平御览》引《诸葛亮集》（一手）
**性质**：写给外甥诸葛乔（过继给诸葛瑾的养子）的志向教育信

**全文**：
> "夫志当存高远，慕先贤，绝情欲，弃凝滞，使庶几之志，揭然有所存，恻然有所感。忍屈伸，去细碎，广咨问，除嫌吝，虽有淹留，何损于美趣，何患于不济。若志不强毅，意不慷慨，徒碌碌滞于俗，默默束于情，永窜伏于凡庸，不免于下流矣。"

**核心论点**：
1. **志当存高远** → 立志的高度决定人生的高度
2. **慕先贤** → 以先贤为参照系
3. **忍屈伸** → 韧性与灵活性
4. **广咨问，除嫌吝** → 开放心态

### 1.6 《便宜十六策》
**出处**：《隋书·经籍志》著录（部分学者认为后人整理，但思想与诸葛亮一脉相承）
**性质**：治国理政的系统方法论，16篇

**十六策篇目**：治国、君臣、视听、纳言、察疑、治人、举措、考黜、治军、赏罚、喜怒、治乱、教令、斩断、思虑、阴察

**核心引文**：
> "治国之政，其犹治家。治家者务立其本，本立则道生。"（治国策）
> "赏罚不明，则善恶无由而见。"（赏罚策）
> "明君纳言，以通下情。"（纳言策）
> "思虑之政，谓不轻举妄动。"（思虑策）
> "夫用人者，求之失之，不若知之慎之。"（治人策）

### 1.7 《将苑/心书》
**出处**：《诸葛亮集》卷四（真伪有争议，自清代起有学者认为是伪托，但思想与诸葛亮军事理念高度一致）
**性质**：军事领导力理论，50篇

**核心篇目与引文**：
> "兵权者，是三军之司令，主将之威势。"（兵权篇）
> "善将者，必有博闻多智者为腹心，沉审谨密者为耳目，勇悍善敌者为爪牙。"（腹心篇）

**知人七法**（知人篇）：
1. 问之以是非而观其志
2. 穷之以辞辩而观其变
3. 咨之以计谋而观其识
4. 告之以难而观其勇
5. 醉之以酒而观其性
6. 临之以利而观其廉
7. 期之以事而观其信

### 1.8 《答法正书》
**出处**：《三国志·蜀书·诸葛亮传》裴松之注引《诸葛亮集》（一手）
**性质**：回复法正"缓刑弛禁"建议的重要政论

**全文**：
> "君知其一，未知其二。秦以无道，政苛民怨，陈涉大呼，天下土崩，高祖因之，可以弘济。刘璋暗弱，自焉已来有累世之恩，文法羁縻，互相承奉，德政不举，威刑不肃。蜀土人士，专权自恣，君臣之道，渐以陵替。宠之以位，位极则贱，顺之以恩，恩竭则慢。所以致弊，实由于此。吾今威之以法，法行则知恩，限之以爵，爵加则知荣矣。荣恩并济，上下有节。为治之要，于斯而著。"

**核心论点**：
1. **威之以法，法行则知恩** → 法治是恩德的前提
2. **限之以爵，爵加则知荣** → 稀缺性创造价值感
3. **宠之以位→贱，顺之以恩→慢** → 过度宽纵反而导致无序
4. **荣恩并济，上下有节** → 恩威并施的系统治理
