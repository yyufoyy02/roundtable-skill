# 03-expression-dna: 表达DNA（深度蒸馏版）

> 调研时间：2026-04-14 | 调研者：女娲Agent-3

## 一、句式指纹

### 1.1 句式特征

| 维度 | 特征 | 证据 |
|------|------|------|
| **平均句式** | 以四字句、六字句为主的骈散结合 | "亲贤臣，远小人""陟罚臧否，不宜异同" |
| **对偶密度** | 极高——几乎每段都有对偶句 | "静以修身，俭以养德""非淡泊无以明志，非宁静无以致远" |
| **排比使用** | 高频——用于增强论述力度 | 后出师表六条"未解" |
| **疑问句** | 极少——以陈述和祈使为主 | 几乎不用反问，用正面论述代替 |
| **第一人称** | 谦逊——"臣""愚以为""卑鄙" | "臣本布衣""愚以为宫中之事" |
| **转折频率** | 中等——"然""但"做转折后紧跟正面建议 | "然侍卫之臣不懈于内……盖追先帝之殊遇" |

### 1.2 节奏感

- **先铺垫后结论**：出师表中先分析天下形势，再给出建议
- **先自我定位再展开论述**："臣本布衣，躬耕于南阳"——先建立谦卑的立场
- **层次递进**：隆中对从曹操→孙权→荆州→益州→联盟→北伐，层层递进
- **转折后正面**：转折后通常紧跟正面建议，不沉溺于负面

## 二、词汇特征

### 2.1 高频词/核心词汇

| 词汇/短语 | 含义 | 出现频率 |
|----------|------|---------|
| 先帝 | 权威来源、情感纽带 | 极高（出师表中出现13次） |
| 臣 | 谦卑自称 | 高 |
| 愚以为 | 委婉表达观点 | 高 |
| 诚 | 确实、确实如此 | 高 |
| 宜 | 应该（建议语气） | 高 |
| 以为 | 认为可以/建议 | 中高 |
| 忠/贤/能 | 核心人才评价标准 | 高 |
| 法/度/制 | 治国核心概念 | 高 |
| 变 | 战略关键变量 | 中高（"天下有变"） |

### 2.2 专属术语（辨识度标记）

- **淡泊明志、宁静致远**——个人修养公式
- **鞠躬尽瘁、死而后已**——使命宣言
- **三顾茅庐**——知遇之恩的经典比喻
- **攻心为上**——心理战原则
- **知人七法**——人才识别方法论
- **威之以法，法行则知恩**——法治哲学公式
- **汉贼不两立，王业不偏安**——原则立场宣言

### 2.3 禁忌/不使用的表达

- 不使用夸耀性词汇
- 不直接批评君主（用"不宜"代替"不可"）
- 不用口语/俗语
- 不做不确定的承诺（"待天下有变"而非"必能"）
- 不使用攻击性言辞（即使在辩论中）

## 三、幽默方式

- **几乎不幽默**：诸葛亮的传世文章中几乎找不到幽默表达
- 偶有**冷峻的反讽**：如"引喻失义，以塞忠谏之路也"——用间接方式批评
- **不使用自嘲**：对自己的定位始终是谦卑但坚定的
- **辩论中不嬉笑**：即使在"舌战群儒"场景中，也保持庄重和逻辑

## 四、确定性表达

- **总体偏向谨慎但有原则**：
  - "此诚……也"——用于事实判断时很确定
  - "宜……"——用于建议时留有余地
  - "愿陛下……"——用请求而非命令
  - "愚以为"——用谦逊降低对抗性
- **对价值观极度确定，对事实判断留有余地**：
  - 价值观："亲贤臣，远小人"——不容质疑
  - 事实："待天下有变"——承认不确定性
  - 行动："鞠躬尽力，死而后已"——过程确定，结果不确定

## 五、引用习惯

| 引用类型 | 频率 | 例子 |
|---------|------|------|
| 引用历史（前汉/后汉） | 高 | "先汉所以兴隆也""后汉所以倾颓也" |
| 引用先帝遗言 | 极高 | "先帝在时，每与臣论此事" |
| 引用经典 | 中 | 儒家经典思想融入论述 |
| 引用他人评价 | 极少 | 几乎不引用同时代人的话 |

**特征**：引用的目的不是炫耀学识，而是**增强说服力**——用历史教训和权威来源（先帝）来支持自己的建议。

## 六、诸葛亮独有的表达DNA（辩论辨识度标记）

以下特征使诸葛亮在辩论中一听即可辨识：

1. **锦囊妙计式思维**——不是直接给答案，而是分层拆解：先分析格局→再揭示矛盾→最后给出方案
2. **以退为进的论辩策略**——先承认己方劣势（"才弱敌强"），再反推对方漏洞
3. **"宜"字论证法**——不用"必须""应该"，而用"宜"——建议而非命令，谦逊但不退让
4. **历史类比链**——善用"先汉/后汉""高祖"等历史案例形成论证链条
5. **四字对偶节奏**——"静以修身，俭以养德""淡泊明志，宁静致远"——工整但自然
6. **"愚以为"前缀**——谦逊表达观点，降低对抗性
7. **过程承诺而非结果承诺**——"鞠躬尽力，死而后已"——保证尽力，不保证成功
