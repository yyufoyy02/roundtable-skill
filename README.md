# Roundtable (圆桌会议)

召集古今中外人物围绕你的问题展开高强度多轮辩论。

不是给你一个答案，而是让多个真实人物的思维方式在你的问题上碰撞。

## 安装

```bash
npx skills add liangweijian/roundtable-skill -g
```

## 使用

安装后在 Claude Code 中输入：

```
圆桌会议 AI会取代程序员吗
```

### 内置成员

孔子、曹操、特朗普、王安石、罗永浩、苏格拉底、莎士比亚、诸葛亮、马克思、马斯克

### 命令一览

| 命令 | 说明 |
|------|------|
| `圆桌会议` | 展示主菜单 |
| `圆桌会议 <议题>` | 发起完整辩论 |
| `圆桌会议 快速 <议题>` | 快速模式 |
| `圆桌会议 对话体 <议题>` | 对话体格式 |
| `圆桌会议 廷议 <议题>` | 廷议格式 |
| `圆桌会议 @马斯克 @孔子 <议题>` | 指定成员辩论 |
| `圆桌会议 加入 <人名>` | 蒸馏新成员（需要 [huashu-nuwa](https://github.com/alchaincyf/nuwa-skill)） |
| `圆桌会议 移除 <人名>` | 移除成员 |

### 辩论中互动

- `纸条 马斯克：你真的认为AI安全吗？` — 暗中递纸条
- `拍桌子 马克思！那苏联怎么解释？` — 立即打断
- `投票 诸葛亮` — 表达支持
- `闪电` — 限时快攻模式
- `交锋 A vs B：争议点` — 指定1v1对决

## 依赖

- [huashu-nuwa](https://github.com/alchaincyf/nuwa-skill) — 成员蒸馏（加入新成员时自动安装）
