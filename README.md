# Roundtable (圆桌会议)

召集古今中外人物围绕你的问题展开高强度多轮辩论。

不是给你一个答案，而是让多个真实人物的思维方式在你的问题上碰撞。

## 安装

```bash
npx skills add liangweijian/roundtable-skill -g
```

## 快速开始

安装后在 Claude Code 中输入：

```
圆桌会议 AI会取代程序员吗
```

## 核心特性

- **真实人物思维模型** — 每位成员基于深度蒸馏，拥有独立的决策框架、表达DNA和辩论弹药
- **多轮渐进式辩论** — 立论 → 交锋 → 转折点 → 裁决，四幕完整叙事
- **实时用户介入** — 递纸条、拍桌子、投票、指定交锋，随时影响辩论走向
- **三种输出格式** — 结构化（默认）、对话体（小说风）、廷议（古风投票制）
- **三种深度模式** — 完整（3轮）、摘要（1轮+裁决）、快速（一次性输出）
- **智能选人** — 自动根据议题领域匹配成员，确保观点多元性和对抗性

## 自定义成员

通过蒸馏机制加入任意真实人物，存放在本地 `members/custom/` 目录，不会被 git 跟踪。

```
圆桌会议 加入 芒格 乔布斯
```

蒸馏依赖 [huashu-nuwa](https://github.com/alchaincyf/nuwa-skill)，首次使用时自动安装。

## 命令一览

| 命令 | 说明 |
|------|------|
| `圆桌会议` | 展示主菜单 |
| `圆桌会议 <议题>` | 发起完整辩论 |
| `圆桌会议 快速 <议题>` | 快速模式 |
| `圆桌会议 摘要 <议题>` | 摘要模式（立论+裁决） |
| `圆桌会议 对话体 <议题>` | 对话体格式 |
| `圆桌会议 廷议 <议题>` | 廷议格式 |
| `圆桌会议 @马斯克 @孔子 <议题>` | 指定成员辩论 |
| `圆桌会议 加入 <人名>` | 蒸馏新成员 |
| `圆桌会议 移除 <人名>` | 移除成员 |
| `圆桌会议 成员` | 成员管理面板 |

以上命令可自由组合，例如 `圆桌会议 快速 对话体 @孔子 AI伦理`。

### 辩论中互动

| 命令 | 效果 |
|------|------|
| `纸条 马斯克：你真的认为AI安全吗？` | 暗中递纸条，该成员下一轮回应 |
| `拍桌子 马克思！那苏联怎么解释？` | 立即打断，逼迫即兴回答 |
| `投票 诸葛亮` | 表达支持，主持人下一轮引用 |
| `闪电` | 第2轮变为限时100字快攻 |
| `交锋 A vs B：争议点` | 第2轮变为两人1v1对决 |
| `暂停` / `继续播放` | 切换手动/自动播放模式 |

## 项目结构

```
skills/roundtable/
├── SKILL.md                    # 主控文件（命令解析、流程编排）
├── prompts/
│   ├── domain-mapper.md        # 成员匹配与阵营划分规则
│   ├── debate-protocol.md      # 辩论协议（轮次、互动、裁决）
│   └── formats/
│       ├── structured.md       # 结构化格式模板
│       ├── dialogue.md         # 对话体格式模板
│       └── court.md            # 廷议格式模板
└── members/
    ├── {内置成员}/SKILL.md      # 默认成员（git 跟踪）
    └── custom/
        └── {自定义成员}/SKILL.md # 用户添加（git 忽略）
```
